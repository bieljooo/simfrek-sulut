## About Laravel

Laravel is a web application framework with expressive, elegant syntax. Development should be enjoyable and productive, and Laravel helps with the common tasks found in modern web applications.

- Simple, fast routing engine.
- Powerful dependency injection container.
- Flexible session and cache storage back-ends.
- Expressive, intuitive database ORM.
- Database-agnostic schema migrations.
- Reliable background job processing.
- Real-time event broadcasting.

Laravel is accessible, powerful, and built for large, robust applications.

## Learning Laravel

Laravel provides extensive documentation and a rich video library that make it easy to get started and stay productive while building modern applications.

## Stack

- Backend: Laravel 11, PHP 8.2+
- Templating: Blade
- Admin Panel: Filament
- Database: MySQL
- Frontend: JavaScript, Bootstrap 5
- Maps: Leaflet
- Build Tool: Vite
- Notifications: SweetAlert2
- Icons: Tabler Icons
